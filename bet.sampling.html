bet.sampling.adaptiveSampling module¶

This module contains functions for adaptive random sampling. We assume we are given access to a model, a parameter space, and a data space. The model is a map from the paramter space to the data space. We desire to build up a set of samples to solve an inverse problem thus giving us information about the inverse mapping. Each sample consists of a parameter coordinate, data coordinate pairing. We assume the measure of both spaces is Lebesgue. We employ an approach based on using multiple sample chains.

class bet.sampling.adaptiveSampling.maxima_kernel(maxima, rho_D, tolerance=1e-08, increase=2.0, decrease=0.5)¶

Bases: bet.sampling.adaptiveSampling.kernel

We assume we know the maxima of the distribution rho_D on the QoI and that the goal is to determine inverse regions of high probability accurately (in terms of getting the measure correct). This class provides a method for determining the proposed change in step size as follows. We check if the QoI at each of the input_new(k) are closer or farther away from a region of high probability in D than the QoI at samples_old(k). For example, if they are closer, then we can reduce the step_size(k) by 1/2.

Module contents¶

This subpackage contains

  • basicSampling a general class and associated set of

    methods that interogates a model through an interface.

  • sampler requests data(QoI) at a

    specified set of parameter samples.

  • bet.sampling.adaptiveSampling inherits from

    basicSampling adaptively generates samples.
