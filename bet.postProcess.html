bet.postProcess.postTools.sample_highest_prob(top_percentile, P_samples, samples, lam_vol=None, data=None)¶

This calculates the highest probability samples whose probability sum to a given value. The number of high probability samples that sum to the value and the probabilities, samples, volumes, and data are returned. This assumes that P_samples, samples, lam_vol, and data have all be sorted using sort_by_rho().

Parameters:
  • top_percentile (float) – ratio of highest probability samples to select
  • P_samples (:class:’~numpy.ndarray’ of shape (num_samples,)) – Probabilities.
  • samples (:class:’~numpy.ndarray’ of shape (num_samples, ndim)) – The samples in parameter space for which the model was run.
  • lam_vol (:class:’~numpy.ndarray’ of shape (num_samples,)) – Volume of cell associated with sample.
  • data (:class:’~numpy.ndarray’ of shape (num_samples, mdim)) – QoI data from running the model with the given samples.
Return type:

tuple
Returns:

( num_samples, P_samples, samples, lam_vol, data)

bet.postProcess.postTools.sort_by_rho(P_samples, samples, lam_vol=None, data=None)¶

Module contents¶

This subpackage contains

  • plotP plots P and/or volumes of voronoi cells
  • plotDomains plots the data domain D in 2D
  • postTools has tools for postprocessing
