bet.postProcess.plotP.calculate_1D_marginal_probs(sample_set, nbins=20)¶

This calculates every single marginal of the probability measure described by the probabilities within the sample_set object. If the sample_set object is a discretization object, we assume that the probabilities to be plotted are from the input space on the emulated samples (discretization._emulated_input_sample_set._probabilties_local).

This assumes that the user has already run prob_emulated().

Parameters:
  • sample_set (sample_set_base or discretization) – Object containing samples and probabilities
  • nbins (:int or ndarray of shape (ndim,)) – Number of bins in each direction.

Module contents¶

This subpackage contains

  • plotP plots \(P\) and/or volumes (\(\mu\))

    of voronoi cells

  • plotDomains plots the data domain

    \(\mathcal{D}\) in 2D

  • postTools has tools for postprocessing
