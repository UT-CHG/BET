bet.vis.plotDomains.show_data_domain_2D(samples, data, Q_true, true_markers=['^', 's', 'o'], true_colors=['r', 'g', 'b'], xlabel='$q_1$', ylabel='$q_2', triangles=None, save=True, show=True, filenames=None)¶

Plot the data domain D using a triangulation based on the generating samples with a marker for various Q_true. Assumes that the first dimension of data is $q_1$.

  • data (np.ndarray) – Data associated with samples
  • Q_true (np.ndarray) – true data value

  • true_colors (list) – list of colors for Q_true
  • xlabel (string) – x-axis label
  • ylabel (string) – y-axis label
  • triangles (tri.Triuangulation.triangles) – triangulation defined by samples
  • save (boolean) – flag whether or not to save the figure
  • show (boolean) – flag whether or not to show the figure
  • filenames (list) – file names for the unmarked and marked domain plots

This module provides methods for plotting probabilities.

bet.vis.plotP.plot_marginal_probs(P_samples, samples, lam_domain, nbins=20, filename='file', lam_true=None, plot_surface=False, interactive=True, lambda_label=None)¶

This makes plots of every pair of marginals (or joint in 2d case) of input probability measure defined by P_samples on a rectangular grid.

Parameters:
  • P_samples (:class:’~numpy.ndarray’ of shape (num_samples,)) – Probabilities.
  • samples (:class:’~numpy.ndarray’ of shape (num_samples, ndim)) – The samples in parameter space for which the model was run.
  • lam_domain (:class:’~numpy.ndarray’ of shape (ndim, 2)) – The domain for each parameter for the model.
  • nbins (:int) – Number of bins in each direction.
bet.vis.plotP.plot_voronoi_probs(P_samples, samples, lam_domain, nbins=20, plot_surface=False)¶

This makes plots of the joint probabilies of input probability measure defined by P_samples for 2d cases. post_process - is an input that only applies to the 2d case w.r.t. the Voronoi cells.

Parameters:
  • P_samples (:class:’~numpy.ndarray’ of shape (num_samples,)) – Probabilities.
  • samples (:class:’~numpy.ndarray’ of shape (num_samples, ndim)) – The samples in parameter space for which the model was run.
  • lam_domain (:class:’~numpy.ndarray’ of shape (ndim, 2)) – The domain for each parameter for the model.
  • nbins (:int) – Number of bins in each direction.

Module contents¶

This subpackage contains

  • plotP plots P and/or volumes of voronoi cells
  • plotDomains plots the data domain D in 2D
